qflow

Build intelligent workflows and agents with JavaScript.

What is qflow?

qflow is a lightweight JavaScript library for building workflows and agents. It uses nodes for atomic operations and flows to chain them together. It supports asynchronous operations and shared state, making it ideal for complex tasks.

Key Features

  • Node-Based Architecture: Work with built-in nodes such as FileSystemNode, HttpRequestNode, CodeInterpreterNode, SemanticMemoryNode, and TransformNode.
  • AgentNode: Integrates an LLM for reasoning and tool selection, following a Reason→Act→Observe loop.
  • Asynchronous Support: Handle async operations seamlessly within flows.
  • Shared State: Easily share data across nodes without complex plumbing.

Getting Started

To begin using qflow, install the library via npm:

npm install qflow

Then, import and create a simple workflow:

const { Flow, HttpRequestNode } = require('qflow');

const flow = new Flow();
flow.addNode(new HttpRequestNode('https://api.example.com/data'));
flow.run();
